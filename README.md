# kuroneko
ヤマト運輸のステータス取得

## installation
    $ go get -u github.com/zombietan/kuroneko/...  
## usage
    $ kuroneko <伝票番号>  
###### example
```
伝票番号 xxxx-xxxx-xxxx
配達完了
このお品物はお届けが済んでおります。

荷物受付　　　　　　　　　　　　　　　| 03/14 | 12:16 | ＸＸセンター　　　　　　　　　　　　　　| 012345 |
発送　　　　　　　　　　　　　　　　　| 03/14 | 12:16 | ＸＸセンター　　　　　　　　　　　　　　| 678901 |
持戻（ご不在）　　　　　　　　　　　　| 03/15 | 14:46 | ＸＸＹ営業支店　　　　　　　　　　　　　| 234567 |
配達完了　　　　　　　　　　　　　　　| 03/16 | 19:10 | ＹＸＸ営業支店　　　　　　　　　　　　　| 890123 |
---------------------------------------------------------------------------------------------------
```
連番で10件まで取得できます  

    $ kuroneko -s 件数 <伝票番号>
